# 🛡️ Base de Conhecimento em Cibersegurança: Prevenção a Phishing

Este repositório contém a entrega do desafio de projeto de Cibersegurança da DIO. O objetivo foi explorar o uso do **Google NotebookLM** como ferramenta de organização e síntese de informações aplicadas à segurança da informação.

## 🎯 Objetivo
Consolidar conceitos de Engenharia Social e Phishing utilizando inteligência artificial generativa. A premissa foi basear a IA em um escopo restrito de fontes documentais confiáveis, garantindo precisão técnica e evitando alucinações.

## 🔗 Acesso ao Projeto
Você pode interagir com a base de conhecimento gerada no NotebookLM acessando o link público abaixo:
👉 **[Acessar Projeto no NotebookLM](https://notebooklm.google.com/notebook/11149a02-692a-462d-8e04-c71373907b55)**

## 📚 Fontes Utilizadas no Repositório da IA
As seguintes fontes (textos e transcrições de vídeos) foram carregadas para alimentar a ferramenta:
1. **Vídeo:** *Cyber Segurança: O que é Phishing?* (Canal: Senai São Paulo)
2. **Vídeo:** *Todos os Ataques Hackers Explicados em 8 minutos* (Canal: DuarttHub)
3. **Cartilha Oficial:** *Fascículo Phishing e Outros Golpes* (CERT.br)

## ⚙️ Aplicação Prática e Resultados
A partir das fontes carregadas, a ferramenta foi utilizada para gerar resumos técnicos e alertas de segurança. O foco da prática foi simular cenários reais de rotina de suporte técnico, criando materiais claros para orientar os usuários finais a identificarem links maliciosos e prevenirem o vazamento de dados nas empresas.
